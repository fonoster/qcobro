## Hello 👋

This is the home of the **QCobro** documentation. QCobro is a multilingual
AI-voice debt-collections platform — a React operator console backed by a tRPC
API over PostgreSQL, with voice, SMS, and email outreach.

<Note>
  This site is just getting started. Pages here are a starting point we'll
  iterate on.
</Note>
